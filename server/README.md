## Getting Started

First, run the development server:

```bash
(1:40:00)
1. Initiate package in server: npm init -y

2. Install prisma client: npm i prisma @prisma/client

3. Initiate prisma: npx prisma init

4. cd prisma

5. copy seedData folder and seed.ts file from remote

6. Install typescript in server level:
npx tsc --init
npm i -D ts-node typescript @types/node

7. Setting up database by creating models in schema.prisma

8. Create connecting string in .env file

9. Execute prisma
npx prisma generate
npx prisma migrate dev --name init
npm run seed (1:55:00)

10. Install packages
npm i express body-parser cors dotenv helmet morgan concurrently

11. Install dependencies
npm i -D nodemon @types/cors @types/express @types/morgan

12. Create source folder at server/src
```
